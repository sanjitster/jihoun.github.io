/exchange

An exchange represents the exchange of a certain amount of a program units into another program units. In legacy documents it can called "conversion".

/exchange/all/{date} get

get

Fetch the full list of exchanges.

get /exchange/all/{date}

Fetch the full list of exchanges.

  • Request
  • Response

URI Parameters

  • date: (integer )

    Timestamp. If specified, only exchanges updated after this timestamp will be returned.

HTTP status code 200

Body

Type: application/json
