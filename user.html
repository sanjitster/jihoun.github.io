/user/update patch

patch

Update current user information.

patch /user/update

Update current user information.

Secured by oauth_2_0
  • Request
  • Response

Body

Type: application/x-www-form-urlencoded

Form Parameters
  • first_name: (string - maxLength: 128)

    First Name

    Example:

    John
  • last_name: (string - maxLength: 128)

    Last Name

    Example:

    Connor
  • address: (string - maxLength: 255)

    Address

  • zip: (string - maxLength: 10)

    Zip code

  • city: (string - maxLength: 128)

    City

    Example:

    United States
  • state: (string - maxLength: 128)

    State

  • country: (string - minLength: 2 - maxLength: 2)

    Country code in upper case.

    Example:

    US
  • phone: (string - maxLength: 20)

    Phone

  • phone_prefix: (string )

    Phone prefix, including the "+" sign.

    Example:

    +33
  • birth_date: (string )

    Birth Date in the format YYYY-MM-DD

    Example:

    2015-12-31
  • lang: (string - minLength: 2 - maxLength: 5)

    User's favorite Language

    Example:

    en
  • username: (string - maxLength: 32)

    Username

    Example:

    Johnny
  • currency: (string - minLength: 3 - maxLength: 3)

    Currency code in lower case

    Example:

    usd
