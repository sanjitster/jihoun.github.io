/oauth2/token post

post

post /oauth2/token

  • Request
  • Response

Body

Type: application/x-www-form-urlencoded

Form Parameters
  • client_id: required (string )
  • client_secret: required (string )
  • grant_type: required (one of authorization_code,refresh_token,password,social )

    Example:

    authorization_code
  • code: (string )

    used when grant_type=='authorization_code'

  • refresh_token: (string )

    used when grant_type=='refresh_token'

  • redirect_uri: (string )

    used when grant_type=='authorization_code'

  • username: (string )

    used when grant_type=='password'

  • password: (string )

    used when grant_type=='password'

  • provider: (string )

    used when grant_type=='social'

    Example:

    google

HTTP status code 200

Type: application/json
